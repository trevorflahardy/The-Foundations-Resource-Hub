1.  How do you access the second element of an array named `values`?
... Remember that C-style arrays are zero indexed.
a)  values(1);
*b)  values[1];
... Square brackets with index 1 retrieve the second element.
c)  values<1>;

2.  Which line declares an array of five integers named `readings`?
... Specify the type, then the name with a size in brackets.
a)  readings int[5];
*b)  int readings[5];
... Correct syntax uses `int readings[5];`.
c)  int[5] readings;
d)  array<int,5> readings;
